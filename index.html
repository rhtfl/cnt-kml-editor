    Экспорт

    Карта и список объектов: обычный клик — одно выделение; Shift, Ctrl или ⌘ + клик — добавить или убрать объект (в т.ч. из другого слоя для ∪∩−). Повторный клик по единственному выделенному — снять выбор. Порядок выбора важен для «Вырезать». Двойной клик по имени — переименовать. Ручка ⋮⋮ — перенос в другой слой.

    Карта · Яндекс
    Рисование
    Режим «Выбор»: обычный клик по объекту на карте — одно выделение (редактор вершин только у одного полигона/линии). Shift, Ctrl или ⌘ + клик — добавить к выбору (∪∩−, в том числе объекты из разных слоёв). Белые круги — вершины, голубые ромбы — новая вершина на ребре. Рисование — панель сверху.
